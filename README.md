# Voice Translator App

This is a **Voice Translator** application developed using **Flutter** for the frontend and **Python** for the backend. The app provides real-time voice translation between **Bangla** and **Chinese**, enabling seamless communication between speakers of these two languages.

## Features

- **Real-time Translation**: Speak in **Bangla** or **Chinese** and get an instant translation.
- **Two-way Translation**: Supports both **Bangla-to-Chinese** and **Chinese-to-Bangla** voice translations.
- **Simple Interface**: The UI allows easy language switching and voice input with just one button tap.

## Technology Stack

- **Frontend**: Flutter (Dart) for building the mobile user interface.
- **Backend**: Python handles the voice recognition and translation services.
- **API**: The app connects to a backend API developed using Python for processing speech-to-text and translating the content.
